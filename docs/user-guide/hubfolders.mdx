In the **Hub Folder** section, as a user you can view the folders/reports created by BI Hub admin.
This serves as a general Hub for all the users to view the common reports organized by users.

You are allowed to also select and add the folders/reports to **My Folders** section.

> You cannot add/delete/edit the folders here.

You can change the report's display as list/tiles, and perform other report's actions as well.
