## Profile

In the Profile section, you can edit your email id, user group and edit/delete the user profile photo.

> You cannot edit the username.

## Preferences

User can set the Landing Page by selecting the tabs from the drop-down list.

The user will land on the preferred page chosen above.

> This is a user-customizable setting and personal to each user.
