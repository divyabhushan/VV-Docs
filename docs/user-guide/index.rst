Welcome to BI Hub User Guide's Documentation!
===============================================

Who Should Read This Guide?
===========================

This User Guide for the BI Hub application offered by Visual BI is meant for the users for basic understanding of how the platform works.

What is BI Hub?
=================

BI Hub a Visual BI product, is a web-based application which enables users to view and analyze reports built
in various BI platforms in one page, without having to login into those platforms individually. Highlights of this product
includes but not limited to:

 - Smart view and Discovery of BI assets – Discover not accessible assets and request access
 - Unified view of BI assets – A consolidated view of reports from multiple BI platforms in one page.
 - Incredibly easy setup and maintenance – Easy to setup and 100% automated synchronization between BI Hub and BI Platforms.
