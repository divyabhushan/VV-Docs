- In the **Agents** column, the Agent icons next to each username indicate the number of agents (BI platforms) in which the user account is present.
- The **Authentication** column, lists the type of authentication assigned to each user.
- You can select the user and click on the Agent Users tab on the right pane to visualize the BI platforms and the name of the user in that platform.
