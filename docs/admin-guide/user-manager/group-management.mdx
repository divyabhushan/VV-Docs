## Manage User Groups

BI Hub admin must create few Power Users who can perform some predefined activities of the Admin user.
To do so, Admin must first create a customized User Group and enable some admin permissions. Then add users to this User Group.

In the **User Manager** screen, click on **Manage User Groups**.

  <div class="center">
    <Zoom>
      <img alt="Manage User Groups" src={useBaseUrl('doc-images/admin-guide/power-users/pr1.png')}/>
    </Zoom>
  </div>

### Add a new User Group

Click **Add User Group**.
  <div class="center">
    <Zoom>
      <img alt="Add User Group" src={useBaseUrl('doc-images/admin-guide/power-users/add-user-group.png')}/>
    </Zoom>
  </div>

Enter a name for the User Group and click **Create**.

### Manage Permissions

Select the User Group and click **Manage Permissions**.
  <div class="center">
    <Zoom>
      <img alt="Manage Users Groups screen" src={useBaseUrl('doc-images/admin-guide/power-users/select-user-group.png')}/>
    </Zoom>
  </div>

Enable the access for specific tasks in both Report Administration and System Administration sections.

  <div class="center">
    <Zoom>
      <img alt="Manage User Groups" src={useBaseUrl('doc-images/admin-guide/power-users/pr5.png')}/>
    </Zoom>
  </div>

Also grant the Dashboard administration permissions:

  <div class="center">
    <Zoom>
      <img alt="Manage Users Groups screen" src={useBaseUrl('doc-images/admin-guide/power-users/sys-dashboard-admin-permissions.png')}/>
    </Zoom>
  </div>

### Add User ( Power Users ) to the Group

Select the User Group ( Power User ) and click **ADD**.

  <div class="center">
    <Zoom>
      <img alt="Add Users to the User Group" src={useBaseUrl('doc-images/admin-guide/power-users/add-users.png')}/>
    </Zoom>
    <p>Add users to the User Group</p>
  </div>

Now, select the users from the drop-down list and click **ADD**.

  <div class="center">
    <Zoom>
      <img alt="User Groups" src={useBaseUrl('doc-images/admin-guide/power-users/power-users-added.png')}/>
    </Zoom>
    <p>Users added to the User Group</p>
  </div>

### Edit User Group

Select a User Group and click **Edit**. This allows you to rename a User Group and click **Save**.

### Delete User Group

To delete the User Group, select the group and click **Delete**.

> If you delete a User Group, the users added to the group will lose the admin privileges assigned to them.

### Remove User(s) from a User Group

To revoke admin permissions from a User, select the User Group and click **Remove** against the User name.
