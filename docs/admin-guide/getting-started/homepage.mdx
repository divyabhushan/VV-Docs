## 1. Instances Configured

This section lists the instances configured,  number of **Reports** synched, the Agent **Server** URL, the Instance **Port** number, and Instance **Status**. 
The instances are managed from the <Link to={useBaseUrl('docs/admin-guide/agent-manager/agent-manager-main')}>Agent Manager</Link> screen.

## 2. Authentications Configured

This section summarizes the user account types with **Authentication Name**, **Type**, and **Status**. These accounts are managed in the <Link to={useBaseUrl('docs/admin-guide/authenticate/authenticate')}>Authentication</Link> section.

## 3. Components Configured

List of components like Elastic Search, Message Queue, and SMTP that were configured during Server configuration.

## 4. Synchronization Information

This section displays the Synchronization details of users, reports, and/or authentication accounts.
These sync reports are triggered from the **Agent Manager > Instance Manager**, and **Authentication** sections.

- To view the recent activities over a period, click on the calendar icon and provide a date range as input.
- To get a detailed information click on **Sync Users**, or **Sync Reports** or **Sync Authentication**, and the **Synchronization information** window pops up.

:::note
Refer the <Link to={useBaseUrl('docs/admin-guide/admin-utilities/admin-utilities#synchronization-information')}>Administrator utility Synchronization</Link> section.
:::
