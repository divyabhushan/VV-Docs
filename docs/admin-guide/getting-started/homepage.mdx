## 1. Instances Configured

List of instances under the agents being configured, number of **Reports** synched, the Agent **Server** URL, the Instance **Port** number, and Instance **Status** is also displayed.


## 2. Authentications Configured

List of user authentications **Name**, **Type**, and **Status**.

## 3. Components Configured

List of components like Elastic Search, Message Queue, and SMTP that were configured during Server configuration.

## 4. Synchronization Information

Synchronization details for users, reports and authentication. 

- To view the recent activities over a period, click on the calendar icon and provide a date range as input.
- To get a detailed information click on **Sync Users**, or **Sync Reports** or **Sync Authentication**, and the **Synchronization information** window pops up.
